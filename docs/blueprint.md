# **App Name**: Pierre's Electrical Engineering Portfolio

## Core Features:

- Homepage Introduction: Introduce Pierre, a 2nd-year Electrical Engineering student at Toronto Metropolitan University.
- Project Showcase: Dedicated sections to showcase the 'Simple General-Purpose Processor' and 'Self-Driving Toy Car' projects, each including descriptions, images, and associated files.
- Contact Information: Provide a contact form or display contact information, like an email address or a link to the student's LinkedIn profile.
- Resume/CV Upload: Enable Pierre to upload his resume for potential employers to download, and display it as part of the UI.
- AI Project Suggestion Tool: An AI-powered tool that suggests potential future electrical engineering projects based on Pierre's existing skills and interests, expanding his project portfolio using LLM reasoning.

## Style Guidelines:

- Primary color: Midnight Blue (#2C3E50) for a professional and reliable feel.
- Background color: Light Gray (#F0F4F8), desaturated hue of Midnight Blue for a clean and modern background.
- Accent color: Teal (#3498DB) to highlight important information, analogous to the primary but brighter for contrast.
- Headline font: 'Space Grotesk', sans-serif, for project titles and headlines; Body font: 'Inter' for project descriptions and other textual information.
- Use professional and consistent icons, like icons for each project indicating the programming languages or electrical components used. Use feather icons library
- Clean and structured layout with clear sections for introduction, projects, and contact information. Prioritize readability and visual appeal.
- Subtle transition animations when navigating between projects or displaying project details.
